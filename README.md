# Snake Game using Python Tkinter


A simple Snake Game developed using Python and the Tkinter library. The game allows players to control a snake, collect food, and increase their score while avoiding collisions with walls and the snake's own body.

## About Tkinter

Tkinter is Python's standard GUI (Graphical User Interface) library. It is used to create desktop applications with windows, buttons, labels, canvases, and other graphical components.

## Game Features

- Snake movement using keyboard controls
- Food generation
- Score tracking
- Collision detection
- Game Over screen
- Simple and interactive GUI

## Technologies Used

- Python
- Tkinter

## How to Run

```bash
python snake_game.py
```
